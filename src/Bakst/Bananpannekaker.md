# Bananpannekaker

| ⏲️ Tid | 🍽️ Porsjoner      | 👨‍🍳 Vanskelighetsgrad |
| ------ | ----------------- | -------------------- |
| 20 min | 12 små pannekaker | Enkel                |

## Ingredienser

#ingredient

- [x] 2 dl lettkokt [[havregryn]]
- [x] 2 [[Banan|bananer]]
- [x] 2 [[egg]]
- [x] 1 dl [[melk]] (+ litt til)
- [x] noe kreativt

## Steg

1. Bland alt sammen og stavmiks det til en fisefin røre.
2. Stek på ganske god varme med en generøs smørklatt.
3. Server gjerne med brunost og syltetøy.
